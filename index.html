🚀 Getting Started with AI-Powered Functional Test Case Synthesizer

Welcome to the AI-Powered Functional Test Case Synthesizer by rSTAR Technologies – your AI-powered assistant for fast, smart, and structured test case generation.

📁 Step 1: Provide Your Input

  • Excel Upload: Download the template, fill in User Story and Acceptance Criteria columns, then upload.
  • ADO Story ID: Enter your Azure DevOps Work Item ID along with organisation, project, and PAT — the tool will fetch the story and acceptance criteria automatically.
  • Optionally attach a design reference image (Figma screenshot) for UI-aware test cases.

⚙️ Step 2: Generate Test Cases

  • Choose your input mode → Generate Test Cases → View results below.
  • Download the Excel and make any changes if required maintaining the same format.

☁️ Step 3: Upload to Azure DevOps

  • Upload your test cases file generated in step 2
  • Configure ADO upload by providing PAT, organisation, test plan, suite and other required details
  • Click on Upload to ADO
⚡ Agentic Flow — Auto Generate & Upload

Provide a single ADO Story ID and credentials — the tool will automatically generate test cases and upload them to Azure DevOps in one click. A test plan named after the story and a TCS suite will be created automatically.
