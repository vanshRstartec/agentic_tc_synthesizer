🚀 Getting Started

Welcome to the AI-Powered Functional Test Case Synthesizer by rSTAR Technologies — enter a single ADO Story ID and let the tool do the rest.

📋 Step 1: Fill in the Details

  • Enter your ADO User Story ID and PAT Token.
  • Optionally override the auto-generated Test Plan or Suite name — leave blank to use defaults.

⚡ Step 2: Choose a Mode & Run

  • Standard: AI generates test cases fresh from the story details.
  • Reference: Provide a reference test plan and suite — AI uses those existing TCs as stylistic guidance, then generates and reviews new ones.
  • Clone: Same as Reference but skips the review agent and uploads directly — best for fast duplication with a different prompt style.
  • Watch progress in real time via the Live Log.

☁️ Step 3: Review & Download

  • Test cases are automatically uploaded to ADO and linked to the story as children.
  • Review the generated test cases in the results table and download the Excel if needed.
⚡ Agentic Flow — Auto Generate & Upload

Provide a single ADO Story ID and PAT — the tool fetches the story, generates test cases via AI agents, and uploads them to Azure DevOps in one click.
