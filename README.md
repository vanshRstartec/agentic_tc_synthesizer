# agentic_tc_synthesizer

Automates test case generation from user story requirements with high accuracy. Seamlessly uploads them in Azure DevOps Test Plans.

## Setup dependencies
pip install -r requirements.txt

## Run the below 2 commands in separate terminals (project root folder)
Terminal 1: python app.py <br />
Terminal 2: python -m http.server 8000 --directory .
